# Домашнее задание по Bash

Этот репозиторий содержит скрипты для домашнего задания по Bash.

## Список скриптов
- hello.sh: Выводит "Hello, World!".
Пример запуска и вывода:
./hello.sh
Hello, World!
- greet.sh: Приветствует по имени.
./greet.sh
Введите ваше имя:
John
Hello, John!
- check_number.sh: Проверяет знак числа.
./check_number.sh
Введите число:
5
Число положительное.
- countdown.sh: Обратный отсчёт.
./countdown.sh 3
3
2
1
0
- file_manager.sh: Управление файлами и директориями.
./file_manager.sh
Файлы созданы.
Файлы удалены.
Директория удалена.
- sum.sh: Сумма двух чисел.
./sum.sh 5 3
8
- read_file.sh: Чтение файла.
./read_file.sh test.txt
Строка 1
Строка 2
- replace_text.sh: Замена текста в файле.
./replace_text.sh test.txt old new
Замена выполнена.
- backup.sh: Резервное копирование с датой.
./backup.sh source_dir backup_dir
Бэкап завершён.
- math_operations.sh: Математические операции.
./math_operations.sh add 5 3
8

## Инструкции по запуску
1. Клонируйте репозиторий: `git clone <ссылка>`.
2. Сделайте скрипты исполняемыми: `chmod +x *.sh`.
3. Запускайте как указано в примерах выше.
