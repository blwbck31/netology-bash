## Список скриптов

### **hello.sh** — выводит текст

```bash
./hello.sh
Hello, World!
```

### **greet.sh** — приветствие по имени

```bash
./greet.sh
Введите ваше имя:
John
Hello, John!
```

### **check_number.sh** — проверка знака числа

```bash
./check_number.sh
Введите число:
5
Число положительное.
```

### **countdown.sh** — обратный отсчёт

```bash
./countdown.sh 3
3
2
1
0
```

### **file_manager.sh** — операции с файлами и директориями

```bash
./file_manager.sh
Файлы созданы.
Файлы удалены.
Директория удалена.
```

### **sum.sh** — сумма двух чисел

```bash
./sum.sh 5 3
8
```

### **read_file.sh** — чтение файла построчно

```bash
./read_file.sh test.txt
Строка 1
Строка 2
```

### **replace_text.sh** — замена текста в файле

```bash
./replace_text.sh test.txt old new
Замена выполнена.
```

### **backup.sh** — резервное копирование с датой

```bash
./backup.sh source_dir backup_dir
Бэкап завершён.
```

### **math_operations.sh** — базовые математические операции

```bash
./math_operations.sh add 5 3
8
```

## Инструкции по запуску

```bash
git clone <ссылка>
cd <папка_репозитория>
chmod +x *.sh
```

Запускайте скрипты согласно примерам выше.
